# RPG風キャラクター管理システム (AETHERIA)

TypeScriptで作られた、型安全なRPGキャラクター管理システムと戦闘シミュレーターです。
ViteとVanilla TypeScriptを使用しており、高速に動作します。

---

## 🚀 起動方法（ローカルでの実行）

### 1. ターミナル（コマンドプロンプトやPowerShell）を開く
デスクトップにある `rpg-character-manager` フォルダを右クリックし、**「ターミナルで開く」**（またはコマンドプロンプトでこのディレクトリに移動）を選択します。

### 2. パッケージをインストールする（初回のみ）
フォルダ内で以下のコマンドを実行し、ViteやTypeScriptに必要なツールをインストールします。
```bash
npm install
```
*(すでにインストール済みの場合は、この手順をスキップできます)*

### 3. 開発サーバーを起動する
以下のコマンドを実行して、ローカルの開発用サーバーを起動します。
```bash
npm run dev
```

### 4. ブラウザで確認する
コマンド実行後、ターミナルに表示される以下のURLにブラウザでアクセスします。
* **`http://localhost:3000/`** (または `http://localhost:5173/` など)

---

## 🛠️ その他のコマンド

### TypeScriptの型チェック ＆ ビルド
プログラムに型エラーがないかチェックし、本番公開用のファイルを `dist` フォルダに出力します。
```bash
npm run build
```

## 📁 フォルダ構成とファイルの説明

* **`index.html`**：ゲーム画面の骨組み（HTML）
* **`src/style.css`**：ネオンファンタジー風の美しいデザイン（CSS）
* **`src/types/game.ts`**：TypeScriptの「型定義」ファイル。キャラクターの能力値、装備枠、バフ効果などを強固に定義しています。
* **`src/engine/character.ts`**：計算式のロジック。レベルアップ時の成長値や、装備による上昇値、バフ効果（「怪力」で攻撃力+30%など）を掛け合わせた実効ステータスを計算します。
* **`src/engine/items.ts`**：装備できるアイテム（武器、盾、鎧、アクセサリ）のデータベース。
* **`src/main.ts`**：ブラウザ上のボタンが押されたときのイベントや、シミュレーター（攻撃や被ダメージ）の動作をつなぎ合わせるプログラム。
